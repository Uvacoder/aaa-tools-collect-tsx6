## 👋 Getting Started

### Installation

1. Clone this repo

```sh
git clone https://github.com/tszhong0411/tools.honghong.me.git
```

2. Change directories

```sh
cd tools.honghong.me
```

3. Install yarn Packages

```sh
yarn
```
